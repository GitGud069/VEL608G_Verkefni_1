Downloads

Fyrst var downloadað Visual Studio Code og Git Bash til að gera þetta verkefni.

Leiðbeiningar til að stofa git repo í command line interface(cli)

echo "# vefsida" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/GitGud069/VEL608G_Verkefni_1.git
git push -u origin main

Hvernig maður commitar á repoið í cli

git add .
git commit -m "skilaboð"
git push -u origin main

Vefsýður code án template

Ég byrjaði að gera vefsíðuna án template því ég fattaði ekki að ég átti að nota það.
Til þess að læra á HTML5 notaði ég Youtube Myndband sem kennir manni á HTML5 á um klukkutíma.

Vefsýðu með template

En þegar ég vissi að maður átti að nota template var því downloadað af HTML5up
Ég valdi Hyperspace því það leit vel út og það virtist mögulegt að byggja upp forritið fyrir mögulega fleiri verkefni.
Svo notaði ég copy-paste til að færa efnið út úr gömlu vefsýðunni yfir á Hyperspace template og færði nokkra hluti inn í Hyperspace file-ið.
Svo notaði ég "index.html" sem Heimasíðu, "generic.html" sem Ferilskrá og "elements.html" sem Vefsíðugerð og aðferðir.

Myndarleit

Eftir að copy-pasta yfir á nýja template fór ég að setja myndir inn forsýðuna og setja link til sýðuna sem ég fékk þær frá með að tengja myndirnar þannig að þegar maður smellir á þær fer maður á vefsýðuna sem þær komu frá.

Github og Git Bash

Til að klára verkefnið þarf að setja nýju vefsýðukóðan með templatinu inn á Github. Til að gera það reyndi ég að nota Git bash en það virkaði ekki, í staðin áhvað ég að nota fara inn á Github og velja add file > upload file. Eftir það fekk ég hjálp í tímanum með Git bash og notað var þessar skipanir.
"cd VEL608G_Verkefni_1/",
"git init",
"git branch -M main",
"git remote add origin https://github.com/GitGud069/VEL608G_Verkefni_1.git"
"git push -u origin main"
Svo til að purfa hvort þetta virkaði var notað
"git add ."
"git commit -m "Test?" "
"git push"
Þannig tengdist Github við file og Visual Studio Code þannig breytingar gerðar í VS Code getu fljótlega verið færðar yfir á Github.
